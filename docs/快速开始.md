# 快速开始

## 初始化

在仓库根目录运行：

```powershell
.\bootstrap.ps1
```

脚本会生成：

- `my-vault/`
- `execution-platform/`
- `execution-platform/projects/`
- `execution-platform/logs/`
- `execution-platform/outputs/`
- `my-skills/`
- `workspace.code-workspace`

## 第一次检索

```powershell
python .\execution-platform\platform-core\scripts\search_kb.py "连续变量 Meta 分析"
```

## 第一次创建任务

```powershell
python .\execution-platform\platform-core\scripts\create_analysis_project.py "demo continuous meta"
```
