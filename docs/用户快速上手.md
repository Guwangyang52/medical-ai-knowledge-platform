# 用户快速上手

本文面向第一次下载本项目的用户，目标是在本地生成一个可被 Obsidian、Codex、Claude Code 使用的医学科研 AI 知识库与执行平台。

## 1. 初始化

在项目根目录运行：

```powershell
.\bootstrap.ps1
```

按提示输入安装目录，例如：

```text
<YOUR_INSTALL_DIR>
```

初始化后会生成：

```text
<YOUR_INSTALL_DIR>/
├─ my-vault/
├─ execution-platform/
├─ my-skills/
└─ workspace.code-workspace
```

## 2. 打开工作区

用 VS Code 打开：

```text
workspace.code-workspace
```

工作区包含：

- `my-vault`：Obsidian / LLM 知识库
- `execution-platform`：任务执行平台
- `my-skills`：用户后续沉淀的技能

## 3. 打开 Obsidian

用 Obsidian 打开：

```text
my-vault/
```

先读：

```text
首页.md
CLAUDE.md
wiki/index.md
```

如果你希望在 Obsidian 内部用 Claudian 辅助整理 raw 资料，可先看：

```text
docs/Obsidian与Claudian安装.md
```

## 4. 理解 raw 和 wiki

```text
raw/   原始资料层
wiki/  整理后的结构化知识层
```

用户可以直接把资料放入：

```text
my-vault/raw/
```

例如：

```text
my-vault/raw/analyses/我的Python分析项目/
my-vault/raw/analyses/我的R分析项目/
my-vault/raw/papers/某篇论文.pdf
```

然后让 Claudian / Claude / Codex 按 `docs/Claudian整理工作流.md` 整理成 wiki。

## 5. 第一次检索

在 `execution-platform` 中运行：

```powershell
python .\platform-core\scripts\search_kb.py "连续变量 Meta 分析" --json
python .\platform-core\scripts\search_kb.py "网状 Meta 分析 BUGSnet SUCRA" --json
python .\platform-core\scripts\search_kb.py "Python 表格数据摘要" --json
python .\platform-core\scripts\search_kb.py "Python 神经网络 MLP 分类预测" --json
```

## 6. 第一次校验

```powershell
python .\platform-core\scripts\validate_kb.py --json
```

期望看到 starter 中的模板状态为 `ok`。

## 7. 第一个 Python 执行任务

```powershell
python .\platform-core\scripts\create_analysis_project.py "python demo" --template-dir "..\my-vault\wiki\analyses\Python表格数据摘要" --copy-template-code --task-id python-demo
Copy-Item "..\my-vault\wiki\analyses\Python表格数据摘要\data\sample_table.csv" ".\projects\python-demo\data\sample_table.csv"
python .\platform-core\scripts\run_python_task.py ".\projects\python-demo" --script "src/analysis.py"
```

运行后查看：

```text
projects/python-demo/output/
projects/python-demo/logs/
projects/python-demo/report/result_summary.md
projects/python-demo/run_manifest.yaml
```
