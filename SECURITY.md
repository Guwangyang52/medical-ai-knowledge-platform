# 安全与隐私策略

## 支持版本

当前项目处于 alpha 阶段。正式版本发布前，安全修复默认面向 `main` 分支。

## 数据安全

本项目是本地优先的科研工作流框架。用户需要自行确保私有数据只保存在本地。

请不要提交：

- 患者数据。
- 私有科研数据。
- 未授权论文或书籍。
- 凭证、token、API key。
- `config/paths.yaml` 等本地路径配置。

## 报告问题

如果发现安全或隐私问题，请尽量使用 GitHub Security Advisory 私下报告。若只能提交 issue，请避免贴出敏感数据。

## 示例数据策略

公开示例应使用：

- 合成数据。
- 明确许可可再分发的公开数据。
- 小体积、可人工检查的文件。

每个可复用 raw 示例都应包含 `SOURCE.md`。
