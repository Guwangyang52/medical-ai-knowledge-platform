# Raw Source Material

`raw/` is the source-material layer of this vault.

Put original papers, datasets, scripts, screenshots, project folders, and rough notes here before compiling them into structured wiki pages.

Recommended structure:

```text
raw/
├── papers/
└── analyses/
```

Rules:

- Keep raw material traceable.
- Add `README.md` and `SOURCE.md` for reusable analysis projects.
- Do not put private, sensitive, or unauthorized material into a public repository.
- Compile reusable knowledge into `wiki/`, rather than treating raw folders as final documentation.
