# 原始资料清单

本目录是开源安全示例。作者本地原始项目中可能还包含 PDF、Word、RStudio 状态文件和大体积输出图，但这些不适合直接进入 GitHub starter。

## 已保留

| 文件 | 用途 |
|---|---|
| `code/analysis.R` | 网状 Meta 分析原始 R 脚本示例 |
| `data/nma_sample_data.csv` | arm-based 示例数据 |
| `output/README.md` | 输出类型说明 |
| `notes/README.md` | 原始整理笔记入口 |
| `SOURCE.md` | 来源与授权边界 |

## 未放入 GitHub starter

| 类型 | 原因 |
|---|---|
| PDF 文献 | 可能涉及版权或再分发限制 |
| Word 文档 | 可能包含个人整理内容或版权材料 |
| TIFF 输出图 | 体积大，且属于运行产物 |
| `.Rhistory` | 本地会话历史 |
| `.Rproj.user/` | RStudio 本地状态 |

用户在自己的私有知识库中可以保留这些原始文件，但公开仓库应先做授权和隐私检查。
