# Data

`nma_sample_data.csv` uses arm-based NMA format.

Required columns:

- `Study`
- `Treatment`
- `Number.of.patients`
- `Number.of.remissions`
- `Number.of.relapses`
- `Number.of.adverse.event.related.withdrawals`
- `Cumulative.dose.of.steroid_Mean`
- `Cumulative.dose.of.steroid_SD`

Each row represents one treatment arm in one study.
