# Source

This raw project is included as a public teaching-style example for the starter vault.

## Data

`data/nma_sample_data.csv` is converted from the example spreadsheet used in the author's local NMA learning project.

Before publishing a derived project, verify that the data source permits redistribution. If redistribution is unclear, replace the file with synthetic or explicitly open data.

## Code

`code/analysis.R` is adapted from the local BUGSnet NMA template and is intended as a reusable scaffold.

## Excluded from GitHub Starter

The following local files are intentionally not included:

- PDF papers or tutorials.
- Word documents.
- RStudio `.Rproj.user/` state.
- `.Rhistory`.
- Large generated TIFF plots.
