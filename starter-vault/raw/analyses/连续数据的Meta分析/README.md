# 连续数据的 Meta 分析 raw 项目

这是一个可公开的最小原始项目示例，用来展示 raw 项目如何整理成 `wiki/analyses/连续数据的Meta分析/`。

## 内容

- `data/sample_continuous_meta.csv`：合成示例数据。
- `code/analysis.R`：可从任务目录运行的连续结局变量 Meta 分析脚本。
- `output/README.md`：输出文件说明。
- `notes/README.md`：整理笔记入口。

## 整理目标

这个 raw 项目被编译为：

```text
wiki/analyses/连续数据的Meta分析/
```

wiki 版本包含 `README.md`、`manifest.yaml`、`code.R`、示例数据和输出说明，供 Codex / Claude Code / MCP 检索与复用。
