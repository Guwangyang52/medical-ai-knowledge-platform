## 本章可沉淀的知识点

神经网络章节适合整理成以下知识单元：

- 人工神经网络的基本结构：输入层、隐藏层、输出层。
- 分类任务中的 MLP：输出类别概率，再转成预测类别。
- 回归任务中的 MLP：输出连续预测值。
- 特征标准化：神经网络通常对变量尺度敏感，建模前应标准化。
- 训练/测试划分：测试集不参与训练和调参。
- 超参数：隐藏层结构、学习率、正则化强度、最大迭代次数。
- 评估：分类准确率、混淆矩阵、分类报告；回归可用 RMSE、MAE、R2。
- 进阶实现：scikit-learn 适合入门和传统表格数据；Keras/TensorFlow 适合更深层网络。

## 可写入 wiki 的整理结论

1. 医学表格数据中的神经网络模板，应先从二分类预测入手。
2. 入门模板不应强依赖深度学习框架，否则新用户难以跑通。
3. 若用户数据量较小，神经网络不一定优于逻辑回归、随机森林或 SVM。
4. 报告模型时不能只报准确率，还应报告混淆矩阵，并根据医学问题补充敏感度、特异度、AUC 等指标。
5. 若使用 Keras，应记录随机种子、框架版本、训练轮数、验证集策略和硬件环境。

## 可引用的小片段式代码结构

以下是开源安全的通用结构，用于帮助 LLM 识别神经网络流程，不是原书代码复制：

```python
model = MLPClassifier(hidden_layer_sizes=(16, 8), alpha=0.001, max_iter=800)
model.fit(X_train_scaled, y_train)
predicted = model.predict(X_test_scaled)
```

Keras 迁移时可使用类似结构：

```python
model = keras.Sequential([
    layers.Dense(64, activation="relu"),
    layers.Dense(1, activation="sigmoid"),
])
model.compile(optimizer="adam", loss="binary_crossentropy", metrics=["accuracy"])
```
