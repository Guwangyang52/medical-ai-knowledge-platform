# raw 到 wiki 整理说明

目标 wiki：

```text
wiki/analyses/Python-神经网络/
```

## 整理步骤

1. 只阅读第 15 章神经网络相关材料。
2. 识别核心任务类型：分类、回归、调参、标准化、模型评估。
3. 将原始脚本拆解为可复用流程：
   - 读取数据
   - 划分训练集和测试集
   - 标准化特征
   - 训练 MLP 模型
   - 输出准确率、混淆矩阵和分类报告
4. 用开源安全示例替代原书代码全文。
5. 在 `manifest.yaml` 中写清：
   - `language: Python`
   - `main_code: code.py`
   - 依赖包
   - 输入要求
   - 输出文件
   - LLM 适配规则

## 神经网络部分重点抽取

整理时建议重点抽取以下内容：

- sklearn 版本：适合传统表格数据和入门演示。
- Keras 版本：适合深层网络和更复杂的模型结构。
- 分类任务：关注准确率、混淆矩阵、敏感度、特异度、AUC。
- 回归任务：关注 RMSE、MAE、R2。
- 预处理：神经网络通常需要标准化，且只能在训练集拟合预处理器。
- 调参：隐藏层大小、正则化强度、学习率、迭代次数、早停策略。

公开版本可以保留极短代码片段帮助用户理解接口，但不要复制原始配套代码全文。
