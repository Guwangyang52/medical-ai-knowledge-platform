# Python 神经网络方法说明

## 1. 什么时候适合用神经网络

神经网络适合处理非线性关系较强、变量之间存在复杂交互的预测任务。医学科研中常见用途包括：

- 疾病二分类预测
- 预后风险分层
- 多分类诊断辅助
- 连续结局预测

但神经网络不是默认最优选择。对于样本量较小、变量数较少、解释性要求高的医学研究，应先与逻辑回归、惩罚回归、随机森林、SVM 等模型比较。

## 2. 标准流程

```text
明确结局变量
-> 清理缺失值和异常值
-> 划分训练集/测试集
-> 仅在训练集拟合预处理步骤
-> 训练神经网络
-> 在测试集评估
-> 输出模型指标和可解释报告
```

关键原则：测试集不能参与标准化器拟合、特征选择、调参或阈值选择。

## 3. 入门实现：纯 Python MLP

本模板的 `code.py` 使用纯 Python 实现一个最小单隐藏层 MLP。它的目标是让用户先跑通执行平台：

- 不需要安装 scikit-learn
- 不需要安装 TensorFlow
- 不需要真实医学数据
- 可以生成标准输出文件

输出包括：

```text
output/metrics.json
output/confusion_matrix.csv
output/training_loss.csv
```

## 4. 迁移到 scikit-learn

当用户已经安装 scikit-learn，并且数据是表格型特征时，可迁移到：

```python
from sklearn.neural_network import MLPClassifier

model = MLPClassifier(
    hidden_layer_sizes=(16, 8),
    activation="relu",
    alpha=0.001,
    max_iter=800,
    random_state=42,
)
```

推荐同时使用：

```python
StandardScaler
train_test_split
confusion_matrix
classification_report
```

## 5. 迁移到 Keras / TensorFlow

当任务需要更深层网络、dropout、batch normalization 或 GPU 训练时，可迁移到 Keras / TensorFlow。

二分类常见输出层：

```python
layers.Dense(1, activation="sigmoid")
```

常见损失函数：

```python
binary_crossentropy
```

多分类常见输出层：

```python
layers.Dense(n_classes, activation="softmax")
```

常见损失函数：

```python
sparse_categorical_crossentropy
```

## 6. 医学研究报告建议

分类任务至少报告：

- 样本量和训练/测试划分比例
- 正负类别数量
- 准确率
- 混淆矩阵
- 敏感度
- 特异度
- AUC 或 F1-score

如果用于论文或正式报告，还应补充：

- 数据来源和纳排标准
- 缺失值处理
- 特征选择方法
- 内部/外部验证方式
- 与传统模型的对照
- 可解释性分析

## 7. 常见问题

- 样本量很小却使用复杂网络，容易过拟合。
- 先对全数据标准化再划分训练/测试，会产生数据泄漏。
- 只报告训练集准确率，没有实际泛化价值。
- 类别不平衡时，准确率可能误导，应报告敏感度、特异度、AUC、F1。
- 没有固定随机种子时，结果难以复现。
